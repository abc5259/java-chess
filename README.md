## 기능 목록

### 체스판
- 체스판은 칸으로 이루어져 있다.

### 칸
- 칸은 넘버링(1~8)과 레터링(a~h) 값으로 구성된다.
 
### 체스 기물
- 체스 기물의 종류는 킹, 퀸, 룩, 비숍, 나이트, 폰이 있다.
- 각각의 기물은 이동 가능한 범위와 경로를 알고있다.
- 각각의 기물은 색을 가지고 있다.

### 체스 기물 핸들러
- 체스 기물의 위치를 조정한다.
